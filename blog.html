What's the difference between map, filter, foreach, and find?

Answer:
map creates a new array by calling a function for every array element.
filter creates a new array filled with elements that pass a test provided by a function.
forEach provides a callback function once for each element in an array in ascending-index order.
find returns the value of the first element that passes a test.

What's the difference between local storage, session storage, and cookies?

Answer:
Local storage: It allows for larger amounts of data to be stored, but the data is not deleted when the browser is closed. Local storage is useful for storing data that the user will need to access later, such as offline data.

Session storage: is similar to cookies, but the data is only stored for the current session. This means that the data will be deleted when the user closes the browser. Session storage is useful for storing data that is sensitive, such as login credentials.

Cookies: are the oldest and most well-known mechanism. They are simple to use and well supported by browsers. However, they are limited to 4KB of data and are often used to store data that is not sensitive, such as user preferences.

Explain quizTimer:

Answer:
If start quiz button is clicked then quizTimer function will be called and this function shows the time Countdown in quiz page.

The 'quizTimer' function is responsible for handling the countdown timer in the quiz app. The function takes a single argument, 'dismiss', which is a boolean value that is used to determine whether to clear the interval timer or not.

Inside the function, an interval timer is created using the 'setInterval' method, which runs every second to update the timer on the screen. The current count of the timer is first converted into minutes and seconds, which are then displayed on the screen.

Additionally, the function also changes the color of the timer text to red after every 60 seconds, as a visual indicator to the user. Finally, the 'clearInterval' method is called to stop the timer when the 'dismiss' parameter is true.

Explain chooseQuiz:

Answer:
This function will take given answer and index as parameter. Then it will check the answer with correct answers and then call the displayAnswers function and pass the answers as parameter to show which answer is selected.

The chooseQuiz function takes two arguments: index and givenAns.

The index argument is an integer that represents the index of the quiz question in the quizData array that the user has chosen to answer.

The givenAns argument is the answer that the user has provided for that particular question.

The function first checks if the user has already answered the question by searching for an object in the answers array that has the same id as the current quiz question. If such an object is found, the function updates the answer for that question in the answers array.

If the user has not answered the question before, the function adds a new object to the answers array with the id and givenAns properties of the current quiz question.

Finally, the function calls the displayAnswers function with the answers array as an argument, which displays all the quiz questions with their corresponding answers.
